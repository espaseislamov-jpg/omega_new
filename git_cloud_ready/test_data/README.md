# Diagnostic snapshot

These files are the saved output for sample `O1_1110012204`.

- `processed_chromatogram.csv` contains the signal and intermediate columns.
- `detected_peaks.csv` and `matched_targets.csv` capture peak assignment.
- `omega_debug.csv` and `omega_summary.json` capture the calculated result.

This is a smoke-test fixture, not the full regression corpus and not a raw
three-header-line input file accepted directly by the GUI loader.
