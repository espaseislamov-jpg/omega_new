Omega v1

Что внутри:
- Omega_v1.exe
- reference_targets_reverted_c22fixed.json
- chromatogram_gui_settings.json

Как установить:
1. Запустить Omega_v1_setup.exe.
2. Дождаться окончания установки.
3. Открывать программу через ярлык на рабочем столе или в меню Пуск.

Как запустить portable-версию:
1. Открыть папку Omega_v1.
2. Запустить Omega_v1.exe.

Что умеет:
- открывает обычные CSV и CHROMTAB.CSV;
- показывает общую хроматограмму, zoom-окна, batch и таблицу пиков;
- считает Omega-3 по встроенному алгоритму;
- работает без установленного Python.

Важно:
- reference JSON и settings JSON должны лежать рядом с exe;
- если антивирус спросит разрешение на запуск нового exe, это ожидаемо для локально собранного приложения.
